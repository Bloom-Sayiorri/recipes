A full-stack recipe sharing platform built with:

- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

Users can sign up, log in, create recipes, save favorites, and manage profiles. Admin users can manage all users.

---

## 🚀 Features

### 👤 Users
- Signup / Login with JWT authentication
- Profile dashboard
- Update profile
- View favorite recipes

### 🍲 Recipes
- Create recipes
- View all recipes
- View single recipe details
- Save favorites

### 🛡️ Admin
- View all users
- Manage user roles (admin/user)
