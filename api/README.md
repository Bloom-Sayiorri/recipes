# 📘 BACKEND README (Express App)

A RESTful API built with Node.js and Express for managing users, authentication, recipes, and favorites.

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt.js
- dotenv
- CORS

---

## 📦 Features

- User authentication (signup/login)
- JWT-based authorization
- Role-based access (admin/user)
- CRUD operations for users
- CRUD operations for recipes
- Favorites system
- Secure password hashing

---

## 🛠️ Installation

```bash
git clone <backend-repo-url>
cd api
npm install
```

## 🚀 Running the App

```bash
npm run dev
```
