# 🍽️ Recipe Share - Frontend

A React-based frontend for the Recipe Share application. Users can sign up, log in, create recipes, view recipes, and manage their profiles. Admin users have additional privileges such as viewing all users.

---

## ⚙️ Tech Stack

- React.js
- React Router DOM
- Context API (Auth Management)
- Tailwind CSS
- SweetAlert2
- Fetch API

---

## 📦 Features

- User authentication (Login / Signup)
- Persistent login using localStorage
- Protected routes
- Create, view, and manage recipes
- User dashboard with profile update
- Admin dashboard (view users, admin controls)
- Favorites system (user-based)
- Responsive UI

---

## 🛠️ Installation

```bash
git clone <frontend-repo-url>
cd frontend
npm install
```

## 🚀 Running the App

```bash
npm start
```
